 # Legal Document Analyzer

A local RAG (Retrieval-Augmented Generation) application for analyzing legal documents.

## Features
- Upload legal documents (PDF, DOCX)
- Ask questions about uploaded documents
- Secure local processing (no external APIs)
- Web-based interface

## Tech Stack
- **Backend:** FastAPI (Python)
- **Frontend:** React (TypeScript)
- **Database:** PostgreSQL + ChromaDB
- **LLM:** Local Llama model via llama.cpp
